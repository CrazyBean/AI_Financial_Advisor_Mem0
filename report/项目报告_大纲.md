# 《基于大语言模型记忆层（Mem0）的个性化智能金融理财顾问系统》
**项目报告大纲（待填充）**

## 1. 介绍解决的问题与研究动机
*   **痛点**：传统金融大模型受限于上下文窗口，患有“健忘症”，无法建立长期的用户信任与专属画像。
*   **动机**：利用业界前沿的智能体记忆框架（Mem0），赋予 AI 金融顾问“记忆”，实现金融建议的千人千面。

## 2. 相关工作
*   对比传统基于标签的用户画像。
*   对比标准 RAG（检索增强生成）在处理动态个人特征上的不足。
*   引入大模型记忆层架构理念。

## 3. 模型设计与实现
*   **架构设计**：交互层 -> Mem0 记忆中枢 -> 决策生成层。
*   **代码实现**：详述 `src/advisor.py` 中如何使用向量化技术对自然语言偏好进行隐式存储和特征召回。
